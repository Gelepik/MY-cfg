Hyprland Binds

Hotkeys for efficient workflow in Hyprland WM (Горячие клавиши для работы в Hyprland)

Main Applications (Основные приложения)
Win + Q
Open Terminal (Открыть терминал)
Win + E
Open File Manager (Открыть файловый менеджер)
Win + R
Open App Menu (Открыть меню приложений)
Win + B
Open Firefox (Открыть Firefox)
Win + T
Open Telegram (Открыть Telegram)
Window Management (Управление окнами)
Win + C
Close active window (Закрыть активное окно)
Win + D
Toggle floating window (Переключить плавающий режим)
Win + F
Toggle Fullscreen (Переключить полноэкранный режим)
Screenshots (Скриншоты)
Win + P
Screenshot area to clipboard (Скриншот области в буфер)
NumPad Binds (Бинды на NumPad)
Win + NumPad 1
Open Steam (Открыть Steam)
Win + NumPad 2
Open Heroic Games Launcher (Открыть Heroic Games Launcher)
Win + NumPad 3
Open Osu! (Открыть Osu!)
Win + NumPad 5
Open Discord (Открыть Discord)
Win + NumPad *
Run Binds Script (Запустить скрипт с биндами)

Created for Hyprland users.
